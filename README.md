# FlowSync - The Orchestrated Event Management Platform

FlowSync is a premium event management system designed for the synthetic architect. It moves beyond standard utility to provide a digital curator experience, inspired by architectural fluidity and editorial data design.

## Features

- **Architectural Fluidity**: A UI that feels like physical layers of glass and light.
- **Glassmorphism**: Sophisticated semi-transparent interfaces with backdrop blurring.
- **Tonal Depth**: Hierarchy defined by color shifts rather than borders.
- **Dual-DNA Typography**: Combining Manrope for display and Inter for functional trust.
- **Dynamic Scheduling**: Real-time orchestration of complex event tracks.

## Technology Stack

- **React**: Core UI library.
- **Next.js**: The React Framework for the Web.
- **styled-jsx**: For scoped CSS.

## Getting Started

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.

## Design Philosophy

FlowSync adheres to the "Invisible Hand" of design. Boundaries are felt through tonal transitions, and importance is conveyed through movement and light.

- **Primary Color**: #003aa0 (authority and stability)
- **Base Surface**: #f7f9fb (cleanliness and focus)
- **Rounding**: 1.5rem (modern and friendly)
